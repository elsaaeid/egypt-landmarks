# Project Overview
This repository contains the user interface for the "Egypt Landmarks" website built with Next.js (App Router) and TypeScript.

## Quick Summary
- Framework: Next.js (App Router)
- Language: TypeScript + JavaScript
- Icons: react-icons
- Styling: CSS Modules + globals.css
- App entry: `Museum/app/`

## Useful Commands
- Build for production: `npm run build`
- Run production version locally: `npm start`
- TypeScript check: `npx tsc --noEmit -p .`
- ESLint check: `npm run lint`

## Important Folder Structure
- `Museum/app/` — App Router pages
- `Museum/components/` — Reusable components
- `Museum/public/assets/images/` — Images and assets
- `Museum/styles/` — CSS Modules files

## How to Contribute
1. Start a new branch for the feature/fix.
2. Run tests/lint checks locally.
3. Open a Pull Request to `master` with a description of the changes.
